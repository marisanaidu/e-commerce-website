06 / Technical SEO (I)Aura Artisans
Infrastructure

Crawlability & Foundation

The core React application must be optimized for search engine bots. Currently, Googlebot may see a blank page before the JS executes. We must ensure a static/pre-rendered version is available.

Critical Technical Checklist

ItemPriorityAction Required
XML SitemapCriticalGenerate static sitemap.xml and submit to GSC.
Robots.txtHighOptimize to disallow /cart and /account routes.
Meta TitlesHighIntegrate react-helmet-async for dynamic titles.
Canonical TagsMedDefine self-referencing canonicals for all pages.

SPA SEO Recommendation

Transition to Server-Side Rendering (SSR) via Next.js or use Vite SSG to ensure all product data is present in the HTML source upon page load. This is the #1 technical blocker for Aura Artisans.

08 / On-Page SEO (Categories)Aura Artisans
Conversion

Collection Page Optimization

In e-commerce, category pages are the "money" pages. We must transform the Shop page from a simple grid into a content-rich landing page.

Category Blueprint

  • SEO Header: Add a descriptive H1 (e.g., "Artisan Necklaces: Handcrafted Gold & Silver").
  • Bottom Content Block: 250 words of rich text explaining materials, sourcing, and "Why Aura Artisans."
  • FAQ Schema: Add "How to care for necklaces?" FAQ to the bottom of the category page.
  • Internal Links: Link to individual best-selling products directly from the description.

Data-Driven Growth
